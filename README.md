# 📖 About Project

Smart Contact Manager is a web-based contact management application developed using **Spring Boot** and **Thymeleaf**. It allows users to create an account, log in securely, manage personal contacts, update profile information, reset passwords, and export contacts in different formats through a simple and responsive interface.

# ✨ Features

- 👤 User Registration
- 🔐 User Login
- 🔑 Forgot Password & Reset Password
- 📇 Add New Contact
- 📋 View All Contacts
- ✏️ Update Contact Details
- ❌ Delete Contacts
- 📊 Dashboard with Contact Statistics
- 👤 Update User Profile
- 🔒 Change Password
- 📄 Export Contacts as HTML
- 📑 Export Contacts as CSV
- 📱 Responsive User Interface
